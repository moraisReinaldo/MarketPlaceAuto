Como ligar o projeto: 

-- Abrir pasta e criar dois terminais 

em um, rodar: node src/BackEndJs/server.js 
que liga a conecção com o banco 

rodar npm install
no outro, iniciar o react: npm run dev

executar o banco manualmente copiando o script do arquivo BancoSql.txt para o workbench (Necessárias mudanças nos arquivos server e database dependendo da maquina)
